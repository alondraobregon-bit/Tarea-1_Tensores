# Implementación de la Librería Tensor++ y Red Neuronal en C++

## Descripción

En este proyecto se diseña e implementa una librería en C++ llamada **Tensor++**, inspirada en bibliotecas científicas como NumPy y PyTorch.

El objetivo es desarrollar una clase `Tensor` capaz de manejar tensores de hasta 3 dimensiones (1D, 2D y 3D), realizar operaciones matemáticas avanzadas y ser utilizada en una aplicación práctica de procesamiento de datos.

Como aplicación, se implementa una simulación de una red neuronal que emplea funciones tales como **ReLU** y **Sigmoid**.

---

## Consideraciones

* La librería soporta tensores de hasta 3 dimensiones.
* Se validan las dimensiones para evitar operaciones incompatibles.
* En caso de error, se lanzan excepciones.
* El manejo de memoria se realiza de forma dinámica.

## Compilación y ejecución

### Opción 1: Usando CLion

1. Clonar el repositorio:

```bash
git clone https://github.com/alondraobregon-bit/Tarea-1_Tensores.git
```

3. Esperar a que CMake configure automáticamente.

4. Ejecutar el programa.

### Opción 2: Descarga manual (ZIP)

3. Abrir el proyecto en CLion.

4. Ejecutar el programa.
